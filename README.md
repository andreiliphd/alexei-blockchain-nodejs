# Alexei - Blockchain network on NodeJS
============

Blockchain implemented on NodeJS.

---

## Features
- Block validation
- Submission of data to blockchain


---

## Setup
Clone this repo:

```
git clone https://github.com/andreiliphd/alexei-blockchain-nodejs.git
```

---


## Usage
Just run:
```
node app.js
```
